## README

Motion for React 是一个用于构建流畅、生产级 UI 动画的 React 动画库。支持从基础属性动画到布局、手势和滚动动画。

Motion 独特的混合引擎结合了原生浏览器动画的性能和 JavaScript 的灵活性。

### Motion for React 的优势

Motion 作为生产就绪的动画库，可轻松创建从微交互到复杂手势驱动效果的各种动画。

其声明式 API 与 React 生态无缝集成，动画可直接关联状态和属性。

了解更多：[Motion 官网](https://motion.dev)

### 特性

- 基于属性的动画
- 布局动画
- 手势和滚动驱动的动画
- 硬件加速
- 跨设备手势识别
