## README

TanStack 是面向 Web 开发者的高质量开源工具库。它提供无头、类型安全且功能强大的工具，涵盖路由、状态管理、数据可视化、数据表格等 Web 应用开发所需的核心功能。

### TanStack Query

TanStack Query 是一个强大的异步状态管理和服务器状态工具，专为简化数据获取而设计。它允许您在 TypeScript/JavaScript、React、Vue、Solid、Svelte 和 Angular 应用程序中，无需依赖全局状态即可轻松获取、缓存、更新和管理所有形式的异步数据。

`@tanstack/react-query` 是 TanStack Query 的 React 实现，提供了一套强大且类型安全的工具，用于处理服务器状态和异步数据流。
